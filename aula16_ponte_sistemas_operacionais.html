Do hardware ao sistema operacional

Ideia da aula: perceber que o Windows 7 precisa reconhecer e controlar o hardware usando drivers.

Glossario rapido: SO = sistema operacional. Driver = pequeno software que ajuda o Windows a conversar com uma peca, como rede, som, video, disco ou USB.

Resumo para o caderno

  • Sistema operacional faz a ponte entre usuario, programas e hardware.
  • Driver e um software que ajuda o sistema a conversar com um dispositivo.
  • Sem driver correto, rede, som, video ou impressora podem falhar.
  • Gerenciador de Dispositivos mostra se o hardware foi reconhecido.
  • Boot carrega o sistema que passara a controlar o computador.

Pratica no Windows 7

  1. Abra Painel de Controle > Sistema e Seguranca > Sistema.
  2. Clique em Gerenciador de Dispositivos.
  3. Escolha 4 categorias: Adaptadores de video, Rede, Unidades de disco e Controladores USB.
  4. Anote: dispositivo, funcao e sinal de alerta se houver.
  5. Explique o que aconteceria se o driver de rede falhasse.
Nao instalar ou remover drivers durante a aula.
