⏱️ Aula 5 (Revisão) — Processadores (conceitos): clock, barramento, “tensão” como ideia

Objetivo: entender o básico do processador e termos comuns sem fazer contas.

✅ Resumo para o caderno

  • CPU executa instruções do sistema e dos programas.
  • Clock (frequência) é uma ideia de ritmo/velocidade de trabalho (conceito).
  • Barramento é caminho de dados entre CPU e outras peças.
  • Energia/calor: CPU precisa de energia e gera calor, por isso existe refrigeração.
Glossário rápido: CPU = processador. Clock = ritmo de trabalho da CPU. Barramento = caminho por onde os dados passam. Cooler/dissipador = peças que ajudam a controlar o calor.
