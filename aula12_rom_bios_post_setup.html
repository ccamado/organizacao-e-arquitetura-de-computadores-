ROM, BIOS, POST e Setup

Ideia da aula: entender que o computador tem instrucoes iniciais para testar componentes e iniciar o sistema operacional.

ROM guarda instrucoes basicasBIOS inicia o hardwarePOST testa ao ligarSetup configura opcoes basicas
Glossario rapido: ROM = Read-Only Memory, memoria com instrucoes basicas. BIOS = Basic Input/Output System, sistema basico de entrada e saida. POST = Power-On Self-Test, autoteste que o computador faz ao ligar. Setup = tela de configuracoes basicas da placa-mae.

Resumo para o caderno

  • ROM e uma memoria com instrucoes importantes de inicializacao.
  • BIOS e o sistema basico que ajuda a placa-mae a iniciar o computador.
  • POST e o autoteste feito quando o computador liga: ele verifica se componentes basicos, como RAM e video, estao respondendo antes do Windows iniciar.
  • Setup e a tela de configuracoes da BIOS/placa-mae.
  • Alterar Setup sem saber pode impedir o boot correto.

Pratica no Windows 7

  1. Sem entrar na BIOS: observe a tela inicial quando o PC liga, se possivel.
  2. No Windows 7, abra Iniciar > Painel de Controle > Sistema e Seguranca > Sistema.
  3. Anote processador, memoria instalada e tipo de sistema.
  4. Abra Gerenciador de Dispositivos e observe categorias de hardware.
  5. No caderno, desenhe a sequencia: ligar > BIOS > POST > boot > Windows.
Seguranca: nao alterar configuracoes da BIOS/Setup durante a aula.
