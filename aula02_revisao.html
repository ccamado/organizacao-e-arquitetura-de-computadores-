🛣️ Aula 2 (Revisão) — Barramentos e chipset

Objetivo: entender como as peças “conversam” dentro do computador: barramentos = caminhos; chipset = organizador.

✅ Resumo para o caderno

  • Barramento = caminho por onde passam dados/informações.
  • Chipset = ajuda a organizar a comunicação entre CPU, RAM, disco e portas.
  • Mapa de conversa: CPU ↔ RAM (uso rápido), CPU ↔ Disco (arquivos), CPU ↔ USB (periféricos), CPU ↔ Rede (internet).
Glossário rápido: CPU = processador. RAM = memória temporária. USB = conexão comum para pendrive, teclado e mouse. Chipset = conjunto que ajuda a organizar a comunicação da placa-mãe.
Dica de diagnóstico: PC lento → pensar em disco/RAM. Pendrive não aparece → USB. Internet cai → rede.
