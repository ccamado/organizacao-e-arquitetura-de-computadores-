📌 Aula 9 — Memória RAM: função e por que existe

Ideia da aula: entender a RAM como memória de trabalho temporária do computador. Ela ajuda o processador a trabalhar com programas e dados que estão em uso agora.

⚡ RAM = memória rápida e temporária 🧠 CPU usa a RAM durante o trabalho 💾 Disco/SSD guarda arquivos por mais tempo 🔄 Desligou? RAM perde o conteúdo
Glossário rápido: RAM = Random Access Memory, memória temporária de trabalho. CPU = processador. SSD = unidade de armazenamento rápida, sem disco girando.

✅ Resumo para o caderno

  • Memória RAM é uma memória rápida usada enquanto o computador está ligado.
  • Ela guarda temporariamente programas e dados que estão sendo usados naquele momento.
  • A CPU acessa a RAM para trabalhar com mais rapidez.
  • A RAM é diferente do disco/SSD: o disco guarda arquivos por mais tempo; a RAM guarda dados temporários.
  • Quando o computador é desligado, os dados que estavam apenas na RAM são perdidos.
  • Quando há pouca RAM, o computador pode ficar lento ao abrir vários programas ao mesmo tempo.

🧪 Atividade prática guiada

  1. Abra o Gerenciador de Tarefas: Ctrl + Shift + Esc.
  2. No Windows 7, clique na aba Desempenho.
  3. Anote no caderno a quantidade de RAM instalada.
  4. Observe o uso da memória antes e depois de abrir um programa leve.
  5. Registre: “RAM instalada”, “uso antes” e “uso depois”.
Windows 7: também é possível conferir a RAM em Iniciar > Painel de Controle > Sistema e Segurança > Sistema. Plano B: desenhar CPU, RAM e Disco no caderno.

🌐 Conexão com servidores e Web

Um servidor que hospeda sites precisa de RAM para manter serviços, páginas, banco de dados e processos em execução. Pouca RAM pode deixar o serviço lento.
