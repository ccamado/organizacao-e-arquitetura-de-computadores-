RAV: reforco dos pontos principais

Ideia da aula: recuperar conceitos essenciais sem inserir conteudo pesado novo.

Glossario rapido: CPU = processador; RAM = memoria temporaria; BIOS = sistema basico de entrada e saida; POST = autoteste ao ligar; Boot = inicializacao; Driver = software que ajuda o Windows a usar um dispositivo.

Mapa mental final

  • Placa-mae: conecta.
  • CPU: processa.
  • Cache: acelera acessos frequentes.
  • RAM: memoria temporaria de trabalho.
  • FLASH/Disco: armazenamento.
  • BIOS/POST/Boot: inicio do computador.
  • Windows 7: sistema operacional que controla recursos.

Pratica RAV no Windows 7

  1. Abra Sistema e registre CPU/RAM.
  2. Abra Computador e registre espaco do Disco Local (C:).
  3. Abra Gerenciador de Dispositivos e escolha 3 categorias.
  4. Monte uma tabela: componente, funcao, evidencia no Windows 7.
  5. Responda ao mini-simulado e corrija com o professor.
Entregavel: mapa mental final + tabela de evidencias.
