Sequencia de boot

Ideia da aula: compreender os passos que acontecem entre apertar o botao de ligar e ver o Windows 7 pronto para uso.

1 Energia2 BIOS/POST3 Dispositivo de boot4 Windows
Glossario rapido: Boot = processo de iniciar o computador. BIOS = sistema basico que inicia o hardware. UEFI = versao mais moderna da BIOS. POST = Power-On Self-Test, o autoteste ao ligar.

Resumo para o caderno

  • Boot e o processo de inicializacao do computador.
  • A BIOS/UEFI inicia o hardware e executa testes.
  • O POST verifica componentes basicos.
  • Depois, o computador procura um dispositivo de boot.
  • O sistema operacional e carregado a partir do disco.

Pratica no Windows 7

  1. Sem alterar a BIOS, observe a tela de inicio do computador.
  2. Anote se aparece marca, tecla de Setup ou mensagem de boot.
  3. Abra Computador e identifique o Disco Local (C:).
  4. Desenhe a ordem: botao > BIOS/POST > disco > Windows 7.
  5. Em dupla, ordene cartoes com essas etapas.
Nao alterar ordem de boot no laboratorio sem autorizacao.
