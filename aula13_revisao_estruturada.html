Revisao com checklist

Ideia da aula: organizar o que foi estudado antes da atividade avaliativa.

Glossario rapido: CPU = processador. RAM = memoria temporaria de trabalho. BIOS = sistema basico que inicia o hardware. POST = autoteste ao ligar. FLASH = armazenamento que nao perde dados ao desligar.

Resumo para o caderno

  • Placa-mae conecta os principais componentes.
  • Barramentos permitem comunicacao entre partes.
  • CPU processa instrucoes; cache acelera acessos frequentes.
  • RAM e memoria temporaria de trabalho.
  • FLASH guarda dados sem energia.
  • BIOS/POST participam da inicializacao.

Pratica no Windows 7

  1. Abra Sistema e anote CPU, RAM e tipo do sistema.
  2. Abra Gerenciador de Dispositivos e escolha 3 categorias.
  3. Abra Computador e veja Disco Local (C:).
  4. Monte uma tabela: componente, funcao, onde observei.
  5. Marque quais temas ainda precisa revisar.
Entregavel: checklist completo no caderno.
